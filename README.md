# 🌊 VOLNA — музыка без границ

Онлайн-плеер, который работает **везде без VPN**. Визуал — в духе SoundCloud, движок воспроизведения — в духе Yandex Music. Чистый статический сайт: HTML + CSS + ванильный JS (ES-модули, без фреймворков, без сборки).

## Возможности

- 🌍 **Доступ по всему миру без VPN** — статика на глобальном CDN, без гео-замков и без авторизации.
- 🔗 **Ссылка на плейлист** — состояние очереди кодируется прямо в URL (`#p=…` / `#pl=…`). Отправляйте куда угодно — откроется как есть. Поддержка `navigator.share` и копирования в буфер.
- ▶️ **Воспроизведение плейлистов** — непрерывная очередь, «Моя волна» с бесконечным автодобавлением, шафл, повтор (выкл/все/один), лайки.
- 🎧 **Bluetooth** — звук `<audio>` идёт на активный системный выход (в т.ч. подключённый Bluetooth). Кнопки гарнитуры/магнитолы/локскрина работают через **Media Session API**. Где доступно (Chromium-desktop) — явный выбор устройства вывода (`selectAudioOutput` + `setSinkId`).
- ➕ **Любой трек по прямой ссылке** на аудиофайл.
- 🌗 Светлая/тёмная тема, горячие клавиши, восстановление сессии (играет с того же места).

## Запуск локально

Нужен любой http-сервер (ES-модули не работают через `file://`):

```bash
cd volna-player
python3 -m http.server 4321        # → http://localhost:4321
# или: npx serve .   /   npx vite
```

## Деплой

Полностью статический, все пути относительные — кладётся на любой CDN **без сборки**:
GitHub Pages, Vercel, Netlify, Cloudflare Pages. Просто залейте папку.

## Горячие клавиши

`Space` play/pause · `←/→` перемотка 5с · `↑/↓` громкость · `N/P` след./пред. · `S` шафл · `R` повтор · `M` mute · `L` лайк · `Q` очередь

## Структура

```
index.html
styles/   tokens · base · layout · components · player · responsive
src/
  lib/      util · likes
  data/     catalog          ← демо-каталог (SoundHelix CDN)
  engine/   player · mediaSession · output
  ui/       waveform · playerBar · queue · views · toast
  share/    share            ← ссылки на плейлисты
  main.js
```

> Демо-каталог — свободно доступные треки (SoundHelix). Каталоги Yandex/SoundCloud
> гео-залочены и закрыты авторизацией, поэтому VOLNA даёт их *ощущение* на свободном
> аудио + импорт собственных ссылок.
